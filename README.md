## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/biosci.git
cd biosci
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment

```bash
cp .env.example .env
```

Edit `.env` with your configuration.

### 4. Run development server

```bash
npm run dev
```
